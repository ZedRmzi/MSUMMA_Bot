Welcome to MSUMMA Bot's documentation!
======================================

**MSUMMA Bot** (/lu'make/) is a Python program that runs a discord bot for clubs. This was specifially made for the Michigan State Mixed Martial Arts club but with the goal of making it flexible enough for others to also use. 
*intuitive* API.

Check out the :doc:`usage` section for further information.



.. note::

   This project is under active development.
